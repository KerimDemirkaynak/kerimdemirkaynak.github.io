Installation

  1. Download the released zip file.
  2. Open the Aegisub file location for Windows:
    • Right-click on the Aegisub shortcut on your desktop and select "Open file location."
  3. Drag the locale and dictionaries folders from the downloaded zip file into the directory where Aegisub is installed.
  4. Windows will ask for administrator permission to overwrite the files. Confirm to continue.

That's it! You can now use Aegisub in Turkish.
